# TempVerseFormer: Temporal Modeling with Reversible Transformers


## Requirements
- Python 3.12
- Nvidia GPU, with latest cuda drivers (>= 12.0)

## Setup
1. Execute .sh file (on Windows run via git bash terminal)
``` bash
$ bash setup.sh
```
2. Set the WandB key (WANDB_API_KEY)
